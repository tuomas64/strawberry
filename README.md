These r-scripts were used to analyze the demographic history data of woodland strawberry: 
"The Late Quaternary climate impact on the genome of the woodland strawberry (Fragaria vesca), a perennial herb":

https://www.biorxiv.org/content/10.1101/2024.10.09.617376v1


We acknowledge the assistance of GPT-4o and GPT-5 in the scripting and data analysis phases of our research. This information should be added to the manuscript also.

Genotyping by sequencing (GBS) data was constructed using the pipeline described in GBS_tassel_pipeline.txt
